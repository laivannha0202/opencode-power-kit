# /ui-final-pass — Final UI Quality Pass

## Description

Final quality check before shipping UI. Verifies visual consistency,
edge cases, responsive behavior, accessibility, and animation smoothness.

## Usage

```
/ui-final-pass [target]
```

- `target`: optional path to component or page. Default: full project.

## Examples

```
/ui-final-pass src/components/
/ui-final-pass
```
