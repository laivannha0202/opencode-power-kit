# /image-to-code — Image to Code

## Description

Convert a design image, mockup, or screenshot into working frontend code.
Detects the project's tech stack and generates matching component code.

## Usage

```
/image-to-code <image-path>
```

- `image-path`: path to the design image (PNG, JPG, SVG, Figma export).

## Examples

```
/image-to-code designs/homepage-mockup.png
/image-to-code screenshots/landing-ref.png
```
