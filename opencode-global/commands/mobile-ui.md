# /mobile-ui — Mobile UI Optimization

## Description

Optimize the current UI for mobile devices. Ensures responsive breakpoints,
adequate touch targets (≥44px), proper viewport meta, and mobile-first layout.

## Usage

```
/mobile-ui [target]
```

- `target`: optional path to component or page. Default: current context.

## Examples

```
/mobile-ui src/components/Navbar.tsx
/mobile-ui
```
