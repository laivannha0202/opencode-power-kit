# /redesign-ui — UI Redesign

## Description

Redesign an existing UI component or page from scratch. Produces a complete
new visual design while preserving functionality and data flow.

## Usage

```
/redesign-ui [target]
```

- `target`: optional path to component or page. Default: current context.

## Examples

```
/redesign-ui src/pages/Dashboard.tsx
/redesign-ui
```
