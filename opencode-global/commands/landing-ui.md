# /landing-ui — Landing Page UI

## Description

Design or refine a landing page. Works with hero section, features grid,
call-to-action, testimonials, pricing, and footer.

## Usage

```
/landing-ui [section]
```

- `section`: optional section to focus on (hero, features, cta, pricing, footer).
  Default: full landing page.

## Examples

```
/landing-ui hero
/landing-ui pricing
/landing-ui
```
