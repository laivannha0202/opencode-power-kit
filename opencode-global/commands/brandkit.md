# /brandkit — Brand Kit Generation

## Description

Generate a complete brand kit / design token system for the project.
Produces color palette, typography scale, spacing system, and component tokens
in JSON or CSS custom properties format.

## Usage

```
/brandkit [--format json|css|tailwind]
```

## Examples

```
/brandkit --format tailwind
/brandkit --format json
/brandkit
```
