# /taste-polish — UI Polish & Refinement

## Description

Polish and refine an existing UI component or page. Improves visual consistency,
color harmony, spacing, typography, responsive behavior, and accessibility.

## Usage

```
/taste-polish [target]
```

- `target`: optional path to component or page. Default: current context.

## Examples

```
/taste-polish src/components/Button.tsx
/taste-polish
```
