Tạo đặc tả ngắn gọn (lite spec) cho task hiện tại:
- Goal: 1-2 câu.
- Scope: bullet 3-7 dòng, đúng những gì sẽ làm.
- Out-of-scope: bullet 2-5 dòng, những gì KHÔNG làm.
- Acceptance criteria: bullet 3-5, mỗi cái verify được (test/command/output).
- Files liên quan: list file path dự kiến chạm tới.
- Risks: 1-3 dòng nếu có.

Không code. Chỉ spec. Ngắn đủ để paste vào PR description.
