# v1.5.0 — Power Mode

**Ngày:** 2026-06-08

## Điểm nổi bật

- **13 core agents** — mỗi agent chuyên sâu một lĩnh vực (architecture, debug, QA, security, DB, API, UI/UX, DevOps, release + fullstack autopilot + 3 lite agents)
- **34 commands** — phân loại theo power workflow, safety, build lifecycle, review, DB/API, QA/E2E, DevOps, quality/security, token/tooling
- **Safety guard** — `scripts/opk-command-guard.sh` cảnh báo/chặn lệnh shell nguy hiểm
- **Agent delegation** — build-strong tự động triệu hồi subagent chuyên biệt
- **`/power-build`** — đầu cuối: spec → architecture → build → QA → security → release
- **`/agent-router`** — định tuyến bằng ngôn ngữ tự nhiên tới đúng agent
- **`/tooling-doctor`** — phát hiện công cụ bên thứ ba (rtk, repomix, semgrep, gitleaks, ...)

## Agent Mới (9)

- `architect-strong` — kiến trúc hệ thống, ADR, quyết định thiết kế
- `debug-strong` — debug sâu với phương pháp khoa học, checkpoint
- `qa-strong` — QA/testing, phân tích coverage, regression testing
- `security-strong` — SAST, secret scan, threat model, dependency audit
- `db-strong` — thiết kế schema, migration safety, tối ưu query
- `api-strong` — API contract, OpenAPI, đồng bộ FE/BE
- `ui-ux-strong` — UI/UX review, accessibility, responsive design
- `devops-strong` — Docker, CI/CD, deploy, infrastructure
- `release-strong` — bump version, CHANGELOG, tag, publish

## Commands Mới (7)

- `/agent-router` — định tuyến task tới agent chuyên biệt
- `/ci-fix` — đọc lỗi CI/test/build và sửa an toàn
- `/e2e-flow` — lên kế hoạch và chạy E2E proof với Playwright
- `/release-check` — kiểm tra VERSION/README/CHANGELOG/tag trước release
- `/kit-audit` — kiểm tra cấu trúc opencode-power-kit
- `/power-build` — full workflow: spec→architecture→impl→QA→security→release
- `/tooling-doctor` — phát hiện công cụ bên thứ ba
