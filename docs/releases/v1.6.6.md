# v1.6.6 — MarkItDown Document Tools Opt-in

**Date:** 2026-06-10

## Highlights

- **`opk markitdown install`** — cài Microsoft MarkItDown (Python) qua pipx hoặc pip --user, dry-run trước khi cài, --yes để auto confirm.
- **`opk markitdown status`** — kiểm tra MarkItDown đã được cài trên PATH chưa.
- **`opk md-convert <input> <output> [--force]`** — convert file PDF/DOCX/PPTX/XLSX/HTML/CSV/JSON/XML/ZIP sang Markdown.
- **`opk doc-to-md <input> <output> [--force]`** — alias cho md-convert.
- **`/doc-to-md` OpenCode command** — hướng dẫn agent dùng `opk` wrapper, không tự cài package.
- **Opt-in only** — không auto install trong `opk up`, không MCP default, không vendor source microsoft/markitdown.
- **Safety-first** — không sudo, không curl|sh, không đọc .env/secrets, không overwrite output không có --force.

## Details

### MarkItDown Integration

- `scripts/install-markitdown.sh` (Linux/macOS): detect python3, ưu tiên pipx, fallback pip --user, dry-run plan, --yes flag.
- `scripts/install-markitdown.ps1` (Windows PowerShell): mirror .sh, dùng `& pipx` / `& pip`.
- `bin/opk`: `markitdown)` subcommand — install gọi script, status kiểm tra PATH.
- `bin/opk`: `md-convert|doc-to-md)` — validate input tồn tại, kiểm tra output overwrite (cần --force), refuse sensitive files (.env, .secret, .key, .pem, credential, token).
- `bin/opk.ps1`: parity tương ứng — `'markitdown'` case, `{md-convert,doc-to-md}` case.
- `opencode-global/commands/doc-to-md.md`: agent command documentation — mô tả usage, safety rules.

### Supported Formats

PDF, DOCX, PPTX, XLSX, HTML, CSV, JSON, XML, ZIP (archive containing supported files).

### Safety

- MarkItDown scripts never: auto-install, vendored source, sudo, curl|sh, read .env/secrets.
- MarkItDown scripts never overwrite output without `--force`.
- MarkItDown scripts refuse to convert sensitive files (.env, .secret, credential, token).
- MarkItDown is not installed during `opk up`, bootstrap, or shell startup.
- `bin/opk` and `bin/opk.ps1` reject md-convert if input file doesn't exist or is sensitive.

## Backward Compatible

100% backward compatible. MarkItDown là opt-in, không ảnh hưởng workflow hiện tại. Tất cả tính năng cũ vẫn hoạt động.

### Usage

```bash
opk markitdown install      # Install MarkItDown (pipx preferred)
opk markitdown status        # Check installation
opk md-convert report.pdf report.md      # Convert PDF
opk md-convert doc.docx doc.md --force   # Convert DOCX, overwrite if exists
opk doc-to-md file.html file.md          # Alias
```
