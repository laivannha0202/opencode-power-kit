# v1.6.5 — One Command Update & Cleanup

**Date:** 2026-06-10

## Highlights

- **`opk up` / `opk update` / `opk upgrade`** — một lệnh update toàn bộ kit: `git pull --ff-only`, kiểm tra verify, không stash, không reset.
- **`opk clean`** — dọn artifacts tạm (`GLOBAL_INSTALL_REPORT.md`, `GLOBAL_PACK_REPORT.md`, `OPK_VERIFY_REPORT.md`, `OPK_DOCTOR_REPORT.md`, `RELEASE_NOTES_v*`).
- **`opk up --clean`** — update + clean trong một lệnh.
- **`scripts/cleanup-agent-artifacts.sh`** — script dọn rác có dry-run/apply mode, Trash dir protection.
- **`opk up --install-global`** — option cài global agents/commands/skills sau update.
- **Working tree protection** — `git stash` bị cấm trong update flow, không dùng `git reset --hard`, không xóa thẳng.
- **GitHub Actions CI** — `.github/workflows/verify.yml`: verify trên push và PR.

## Details

### One Command Update

- `bin/opk`: thêm `up|update|upgrade)` case — git pull --ff-only, verify.sh, --clean, --install-global flags.
- `bin/opk.ps1`: parity tương ứng với PowerShell (`'up'` case).
- Update chỉ dùng `git pull --ff-only` — không force push, không stash, không reset.
- `--install-global`: copy agents/commands/skills từ kit vào `OPENCODE_CONFIG_DIR`.
- `--clean`: gọi cleanup script sau update.

### Cleanup

- `scripts/cleanup-agent-artifacts.sh`: dry-run (mặc định) / apply (--yes) mode.
- Dùng Trash dir (`.opk-trash/`) thay vì xóa thẳng — có thể recover.
- Dọn các file artifact: `GLOBAL_INSTALL_REPORT.md`, `GLOBAL_PACK_REPORT.md`, `OPK_VERIFY_REPORT.md`, `OPK_DOCTOR_REPORT.md`, `RELEASE_NOTES_v*`.
- `bin/opk`: `clean)` case.
- `bin/opk.ps1`: `'clean'` case.

### CI & Verification

- `.github/workflows/verify.yml`: chạy trên push và pull_request.
- `verify.sh` / `verify.ps1`: thêm check cho `opk up`, `opk clean`, cleanup script patterns.

## Backward Compatible

100% backward compatible. Update và Clean là optional commands. Không thay đổi behavior của các lệnh cũ.
