# v1.3.0 — Cross-platform

**Ngày:** 2026-06-04

## Điểm nổi bật

- **Hỗ trợ Windows PowerShell** — đầy đủ: `bootstrap.ps1`, `setup.ps1`, `install.ps1`, `verify.ps1`, `doctor.ps1`, `install-global.ps1`
- **`bin/opk.ps1`** — Windows CLI wrapper
- **`bin/opk.cmd`** — Windows CMD shim
- **`bootstrap.sh`** — root installer mới với flags
- **Hỗ trợ zsh** (macOS default shell)
- **4 lệnh opk mới**: `init`, `quick`, `bootstrap`, `one`

## An toàn

- Không sudo, không curl|sh trong bất kỳ script nào
- Windows: chỉ User environment, không system-wide registry
- Tất cả scripts idempotent
- Backup trước khi modify

## Tương thích ngược

100% tương thích ngược với v1.2.0.
