# v1.3.0 — Cross-platform

**Date:** 2026-06-04

## Highlights

- **Windows PowerShell support** — full parity: `bootstrap.ps1`, `setup.ps1`, `install.ps1`, `verify.ps1`, `doctor.ps1`, `install-global.ps1`
- **`bin/opk.ps1`** — Windows CLI wrapper
- **`bin/opk.cmd`** — Windows CMD shim
- **`bootstrap.sh`** — new root installer with flags
- **zsh support** (macOS default shell)
- **4 new opk commands**: `init`, `quick`, `bootstrap`, `one`

## Safety

- No sudo, no curl|sh in any script
- Windows: only User environment, no system-wide registry
- All scripts idempotent
- Backup before modify

## Backward Compatible

100% backward compatible with v1.2.0.
