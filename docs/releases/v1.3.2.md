# v1.3.2 — One-command All-in-one

**Ngày:** 2026-06-06

## Điểm nổi bật

- **`opk one` / `opk go`** — shorthand all-in-one
- **Luồng 4 bước `--all`** — `[1/4] global → [2/4] project → [3/4] fullstack → [4/4] verify`
- **All-in-one one-liner** — một dòng duy nhất cho Linux/macOS/Windows
- **Success banner cuối cùng** — `✅ OpenCode Power Kit all-in-one done. Run: opencode`

## Tương thích ngược

Ngữ nghĩa `opk one` đã thay đổi (global → all-in-one). Dùng `opk quick` hoặc `opk global` cho hành vi cũ.
