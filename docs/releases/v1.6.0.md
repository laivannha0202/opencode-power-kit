# v1.6.0 — Full Auto Permission Mode + Vietnamese Language Lock

**Date:** 2026-06-08

## Highlights

- **Full Auto Permission Mode** — nâng cấp từ safe-first granular allowlist lên `"permission": "allow"`. Agent tự chạy tool/sửa file/bash/test/build mà không hỏi lại permission, phù hợp máy/project cá nhân.
- **Vietnamese Language Lock** — rule bắt buộc cho toàn bộ agents/rules: mặc định trả lời bằng tiếng Việt, giữ tiếng Anh cho code/lệnh/path/log/keyword kỹ thuật.
- **46 agents patched** — tất cả agent files trong `opencode-global/agents/` được thêm Vietnamese Language Lock instruction.
- **Zero dependency** — tính năng chỉ dùng markdown/config, không thêm repo/package/installer mới.
