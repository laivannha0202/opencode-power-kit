# v1.0.0 — Production Release

**Ngày:** 2026-05-25

## Điểm nổi bật

- **Global pack**: 4 agents, 15 commands, 12 skills
- **Cài đặt theo project**: tạo AGENTS.md, OPENCODE.md, .opencode/opencode.json
- **Cài đặt toàn cục**: cấu hình OPENCODE_CONFIG_DIR và PATH
- **Tích hợp BMAD**: cài BMAD Method qua npx
- **Superpowers enabled**: thông qua opencode.json template
- **Token tools**: phát hiện rtk và tokscale
- **Verify** (verify.sh): global config, pack structure, MCP absence, secrets
- **Pack validator** (validate-opencode-pack.py): frontmatter + structure checks
- **Integration test** (integration-test.sh): test project tạm
- **Doctor** (doctor.sh): diagnostic read-only
- **Gỡ cài đặt** (uninstall.sh): restore từ backup
- **CI**: 12 jobs — bash syntax, shellcheck, shfmt, JSON/YAML templates, no-MCP, no-secrets, integration test

## An toàn

- Không có MCP config trong opencode-global/
- Không có secret patterns trong source
- Không curl|sh, không sudo trong install scripts
- Backup trước khi overwrite
