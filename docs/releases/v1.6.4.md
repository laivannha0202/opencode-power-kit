# v1.6.4 — Safety & Compatibility Polish

**Date:** 2026-06-10

## Highlights

- **Power Mode vs Safe Mode** — hai template cho hai chế độ: `opencode.power.json` (full auto permission, dành cho máy cá nhân) và `opencode.safe.json` (safe-first granular, dành cho máy công ty/team).
- **`opk mode` command** — `opk mode show`, `opk mode power`, `opk mode safe` — chuyển đổi giữa Power và Safe mode nhanh chóng.
- **Safety plugin guard** — `templates/plugins/opk-safety-guard.js` — guardCheck function kiểm tra destructive commands trước khi chạy.
- **Normalized command frontmatter** — tất cả command files trong `opencode-global/commands/` được thêm frontmatter chuẩn (`description` + `mode`).
- **`verify.sh` / `verify.ps1` mở rộng** — thêm check cho Power/Safe mode, safety plugin, mode command.
- **No MCP default** — kit không đặt MCP config mặc định.
- **No vendored source** — không bundle third-party source code.

## Details

### Power Mode vs Safe Mode

- `templates/opencode.power.json`: `"permission": "allow"` — agent tự động chạy tool, sửa file, bash, test, build.
- `templates/opencode.safe.json`: granular allowlist — từng tool permission riêng, phù hợp môi trường shared/company.
- `bin/opk`: thêm `mode)` subcommand — `show`, `power`, `safe`.
- `bin/opk.ps1`: parity tương ứng với PowerShell.
- `scripts/install-safety-plugin.sh` + `.ps1`: install/uninstall safety plugin.
- `templates/plugins/opk-safety-guard.js`: guardCheck function.
- README: section "Power Mode vs Safe Mode".

### Command Frontmatter Normalization

- 15 command files trong `opencode-global/commands/` được thêm `description` + `mode` frontmatter.
- `scripts/validate-opencode-pack.py`: thêm validation cho `description` + `mode` trong agent files.
- verify mở rộng: check Power/Safe mode, safety plugin, mode command.

### Script & Tooling

- `scripts/install-safety-plugin.sh` (Linux/macOS): cài safety plugin vào OpenCode config.
- `scripts/install-safety-plugin.ps1` (Windows): mirror PowerShell.
- `verify.sh` / `verify.ps1`: thêm 15+ assertions cho v1.6.4 features.

## Backward Compatible

100% backward compatible. Power/Safe mode là opt-in. Mode mặc định không thay đổi. Tất cả tính năng cũ vẫn hoạt động.
